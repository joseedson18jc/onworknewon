## Wave 4 addendum — server-side auth + API proxy landed

Wave 4 replaced the demo-only client bypass with a real server tier:

- `app/api/auth/{login,logout,me,forgot}.ts` — JWT httpOnly cookie (jose, HS256, 8 h TTL), Argon2id hashing (@node-rs/argon2, OWASP params), 400 ms constant-delay on failed login, forgot-password via Resend with demo-link fallback.
- `app/api/ai/completions.ts` — OpenAI proxy, 30/min + 300/day per-user sliding-window rate limit (Upstash), SSE streaming, never logs prompt/response bodies.
- `app/api/voice/tts.ts` — ElevenLabs TTS proxy (audio/mpeg), 10/min per-user rate limit.
- `app/api/enrichment/surfe.ts` — Surfe person/company enrichment proxy, 20/min per-user rate limit.
- `app/api/_lib/` — shared modules: env validation (Zod), Upstash Redis KV client with in-memory fallback, password hashing, JWT helpers, rate-limit buckets, append-only audit log (90-day KV retention).
- `app/scripts/seed-users.mjs` — Argon2id-seeds Vercel KV from `users.seed.json`.
- `app/.env.example` documents every env var; missing secrets trigger loud warnings in API responses + a demo-mode banner in the UI.
- `app/src/lib/{api,useSession}.ts` — typed fetch wrapper + `useSession` hook. Auto-detects whether `/api/*` is live; falls back to client-only demo so `vite preview` still boots.
- `.github/workflows/ci.yml` + `.gitleaks.toml` — CI runs secret scanning (Gitleaks, custom ElevenLabs + Surfe rules) and typecheck + build on every push.

## Wave 5 addendum — full routes + ⌘K + graph + heatmap + a11y tests

Wave 5 filled in every placeholder from Wave 3 with real implementations,
dropped the bundle critical path below budget via lazy-routing, and
added the first Playwright a11y regression suite.

### Routes shipped

- **`AccountsRoute`** — virtualized ICP table via `@tanstack/react-table`
  + `@tanstack/react-virtual`. 50 accounts (10 real brand seeds +
  40 synthetic mid-tail). Multi-column sort (keyboard-accessible
  headers), full-text global filter, animated slide-in **Account Detail
  Drawer** with decision-makers list, briefings, and deal stats.
- **`CoPilotRoute`** — streaming AI chat over
  `/api/ai/completions?stream=true`. Client-side SSE parser with
  canned-response fallback in demo mode. Starter prompts localised in
  PT / EN / zh-CN. Live-region a11y announcements. Integrates
  `/api/voice/tts` for read-aloud on assistant messages.
- **`CadenceRoute`** — 365-day GitHub-style activity heatmap. 4-stat
  header (total, last-30-day delta, peak, active streak). Click any
  day to drill in.
- **`MarketMapRoute`** — **force-directed decision-maker graph**
  rendered with `d3-force` + React SVG. Per-account stakeholder web,
  stance-coloured nodes (champion / neutral / blocker / unknown),
  influence-weighted node radius, keyboard-navigable, screen-reader
  text fallback via `<details>`.

### Global

- **`CommandPalette` (⌘K / Ctrl+K)** — `cmdk`-powered fuzzy search
  across navigation, accounts (all 50), and actions (theme toggle,
  language switch, sign-out). Opens from the top-bar search button or
  keyboard shortcut. Auto-closes on Escape.
- **`Heatmap` + `ForceGraph`** — reusable chart primitives joining the
  existing `SparkLine` / `DonutChart` / `BarChart` set.
- **Route data-model expansion** — `Account` now carries
  `decisionMakers` + `briefings`; seeded cadence generator produces a
  deterministic 365-day activity series.
